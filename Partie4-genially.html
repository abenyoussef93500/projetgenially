Genially dans Elea (Moodle)

Méthode 1 : Intégration par lien URL (simple)
Cette méthode est la plus rapide et fonctionne avec tous les comptes Genially.
  • Depuis votre Genially publié, cliquer sur « Partager » puis « Lien » et «Toute pesonne ayant le lien»
  • Dans la nouvelle fenêtre, cliquer sur « Copier le lien » pour valider le public désigné
  • Le lien URL est copié dans le presse-papier
  • Se connecter à votre espace Elea ;
  • Accéder au cours dans lequel vous souhaitez intégrer le Genially ;
  • Activer le mode « Édition » (bouton en haut à droite);
  • Cliquer sur « Ajouter une activité ou une ressource »;
  • Sélectionner « URL »;
  • Dans le champ « URL externe », coller le lien de votre Genially;
  • Donner un nom à l'activité (ex : « Quiz interactif - Calcul mental »);
  • Dans « Apparence », choisir « Intégrer » pour afficher le Genially directement dans la page Moodle;
  • Enregistrer et vérifier l'affichage;

Méthode 2 : Intégration par code embed (avancé)
Cette méthode offre plus de contrôle sur l'affichage.
  • Depuis votre Genially publié, cliquer sur « Partager » puis « Insérer » ;
  • Choisir le code iframe et cliquer sur « Copier » ;
  • Dans Moodle, aller dans votre cours en mode « Édition »
  • Ajouter une activité « Page » ;
  • Dans Contenu, cliquer sur « Outils » puis sur « <>Code source »;
  • Coller le code iframe;
  • Enregistrer et visualiser;


Avantages
  • Genially permet de créer des contenus interactifs bien plus attrayants que les ressources statiques natives de Moodle. Cela stimule l'engagement des apprenants. ;
  • Les animations, les effets interactifs et les designs proposés par Genially améliorent considérablement l'esthétique des cours Moodle, souvent perçus comme austèr;
  • Genially génère un code d'intégration (embed) qui s'insère facilement dans Moodle;
  • Si le contenu Genially est modifié, la mise à jour est automatiquement répercutée dans Moodle sans avoir à retoucher le cours;
Inconvénients
  • Si Genially est en panne, hors ligne ou change ses conditions d'utilisation, le contenu devient inaccessible dans Moodle;
  • Moodle ne peut pas tracker les interactions faites dans un Genially (absence de contenu SCORM). Cela limite le suivi pédagogique.;
  • Genially n'est pas toujours conforme aux normes d'accessibilité (WCAG), ce qui peut poser problème pour les apprenants en situation de handicap;
  • Les fonctionnalités avancées de Genially nécessitent un abonnement payant. La version gratuite limite la collaboration et certaines options interactives;
  • Certaines configurations Moodle bloquent les iFrames externes pour des raisons de sécurité. Ce qui peut rendre l'affichage difficile ;
En résumé, c'est une excellente solution pour enrichir visuellement un cours, mais elle ne remplace pas des outils nativement intégrés à Moodle pour le suivi et l'évaluation.
