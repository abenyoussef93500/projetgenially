Comment faire un quiz dans Genially ?

Comment faire un quizz ?
  • Cliquer sur « Créer un genially » depuis le tableau de bord;
  • Choisir " A partir d'un modèle" ;
  • Dans la barre de recherche, taper « quiz » ou « questionnaire »;
  • Parcourir les templates proposés et sélectionner-en un qui correspond à votre discipline;
  • Cliquer sur « Utiliser ce modèle » pour l'ouvrir dans l'éditeur;
💡 Astuce : Privilégier les templates marqués « Interactif » pour bénéficier de fonctionnalités avancées (feedback automatique, score final).

Étape 2 : Personnaliser le contenu
    Une fois dans l'éditeur, vous allez remplacer le contenu générique par vos propres questions.
  • Modifier le titre en cliquant sur le texte du titre et remplacez-le par votre propre titre ;
  • Ajouter des questions :
    • Naviguer entre les diapositives via la barre latérale gauche;
    • Saisisser vos questions;
    • Modifier les propositions de réponse (A, B, C, D);
  • Définir la bonne réponse en suivant les instructions du template;
Étape 3 : Ajouter des animations et transitions
    Les animations rendent le quiz plus dynamique et captivant.
  • Animer l'apparition d'éléments;
    • Sélectionner un élément (texte, image, bouton) ;
    • Cliquer sur l'icône « Animation » dans le menu de droite (icône d'étoile) ;
    • Choisir un effet d'apparition (Fondu, Glisser, Rebond, etc.) ;
    • Ajuster la durée et le délai si nécessaire;
  • Créer des transitions entre diapositives
    • Cliquer sur « Transition » dans la barre du haut;
    • Sélectionner un effet de transition (Fondu, Glisser, Cube, etc.);

Étape 4 : Intégrer des médias (images, vidéos, sons)
  • Ajouter une image :
    • Cliquer sur « Insérer » dans le menu supérieur, sélectionner « Image »;
    • Vous pouvez :
      • Télécharger une image depuis votre ordinateur;
      • Rechercher dans la banque d'images gratuites de Genially;
      • Insérer une URL d'image en ligne;
    • Redimensionner et positionner l'image sur la diapositive;
  • Intégrer une vidéo :
    • Cliquer sur « Insérer Vidéo », coller l'URL d'une vidéo YouTube, Vimeo ou autre plateforme supportée. La vidéo sera lue directement dans le quiz;
  • Ajouter un son ou une musique de fond;
    • Cliquez sur « Insérer » > « Audio », télécharger un fichier audio (MP3 recommandé) ou utilisez la bibliothèque de Genially;
